## hf

### 4/b

$$\cos{10x} = \cos{2x}$$

1. ugyanaz
    $$ k \in Z$$
    $$10x = 2x + k2\pi$$
    $$8x = k2\pi$$
    $$x = \frac{1}{4}k\pi$$
2. másik oldal
    $$k \in Z$$
    $$10x = -2x + k2\pi$$
    $$12x = k2\pi$$
    $$x = \frac{1}{6}k\pi$$

### 4/c

$$\cos{4x} = \sin{3x}$$
$$\cos{4x} = \cos{\frac{\pi}{2} - 3x}$$

1. ugyanaz
    $$k \in Z$$
    $$4x = \frac{\pi}{2} - 3x + k2\pi$$
    $$7x = k2\pi + \frac{\pi}{2}$$
    $$x = \frac{2}{7}k\pi + \frac{\pi}{14}$$

2. másik oldal
    $$k \in Z$$
    $$4x = -\frac{\pi}{2} + 3x + k2\pi$$
    $$x = k2\pi - \frac{\pi}{2}$$
